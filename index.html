How to play

  1. Pick a category (or Mixed) and the number of questions.
  2. You have 12 seconds per question. Faster answers = more points.
  3. Streaks give a multiplier. Use 50/50 or Hint when you need it.

Event organizers can set an event code and a Google Form to collect scores.
